- [ ] 光栅主循环优化
目标：edge function 增量更新，减少像素循环内临时对象分配。

- [ ] 并行化
目标：按 tile 或 scanline 多线程光栅化。

- [ ] 深度与状态优化
目标：完善 Early-Z，减少重复状态计算。

- [ ] 模型矩阵缓存与片段写回收敛（新增）
目标：减少每帧重复矩阵计算与中间片段容器带来的内存带宽开销。
思路：
1. 在 `Object` 中新增 dirty 标记与缓存 `modelMatrix`，仅在 `setPosition/setRotation/setScale/rotate/translate` 后重算。
2. 将当前 `fragments_ -> colorBuffer` 的二次遍历路径改为“可选直写模式”，默认直写颜色缓冲，调试时再保留片段缓存。
3. 若需保留后处理能力，可将片段缓存改成按 tile 局部缓存，避免全屏级全量 append 与随机写。
验收：在相同分辨率和模型下，CPU 帧时间明显下降，且画面结果与当前版本一致。
